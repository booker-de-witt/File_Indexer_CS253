# CS253 Assignment 1 — Versioned File Indexer

A memory-efficient word-frequency indexer that processes large text files using a fixed-size buffer in C++.

## Compilation

```bash
g++ -O2 -std=c++17 -o analyzer rollnumber_firstname.cpp
```

## Usage

```bash
# Word count
./analyzer --file dataset_v1.txt --version v1 --buffer 512 --query word --word error

# Top-K words
./analyzer --file dataset_v1.txt --version v1 --buffer 512 --query top --top 10

# Diff between two versions
./analyzer --file1 dataset_v1.txt --version1 v1 --file2 dataset_v2.txt --version2 v2 --buffer 512 --query diff --word error
```

## Arguments

| Argument | Description |
|---|---|
| `--file` | Input file |
| `--version` | Version name |
| `--buffer` | Buffer size in KB (256–1024) |
| `--query` | Query type: `word`, `top`, `diff` |
| `--word` | Word to look up |
| `--top` | K for top-K query |

For diff queries, use `--file1`, `--file2`, `--version1`, `--version2`.
